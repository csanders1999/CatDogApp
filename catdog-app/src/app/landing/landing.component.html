Our data mining term project is to use several methods to determine if a given user is a dog person or a cat person. We also seek to uncover the general differences in personality between the two classifications. Methods used in determining class include sentiment analysis on tweets containing keywords pertaining to cats or dogs, overall personality analysis of a user's tweets, and image analysis to see if the user posts images of cats or dogs.
